Start with components + signals

  • Use signal() and computed() in a component for strictly local UI.
  • Use for ephemeral stuff that doesn't need to survive navigation.

When to use:

Perfect for simple, component-scoped state that doesn't need to be shared.

Example

@if (isOpen()) {
Workout details go here...
}

Code Implementation:

// Component
export class ComponentSignalsComponent {{ '{' }}
  isOpen = signal(false);

  toggle() {{ '{' }}
    this.isOpen.update((value) => !value);
  {{ '}' }}
{{ '}' }}

<button (click)="toggle()">Toggle details</button>
{{ '@' }}if (isOpen()) {{ '{' }}
  <div>Workout details go here...</div>
{{ '}' }}
