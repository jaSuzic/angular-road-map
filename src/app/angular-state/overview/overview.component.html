What is "state"?

Everything your app needs to remember to render correctly or make decisions.

Common buckets

  • Local UI state: toggles, inputs, dialog open/close, active tab.
  • Derived/computed state: filters, totals, view-models from base data.
  • Server/cache state: workouts, users, pagination; often fetched via HTTP.
  • Session/auth state: current user, token, roles/permissions.
  • Navigation state: route params, query params, router data.
